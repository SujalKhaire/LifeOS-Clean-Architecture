This project is an Android application built **for practicing and demonstrating Clean Architecture, Hilt, Room, and Jetpack Compose**.

The goal of this project is **learning and architecture practice**, not production use.

## Purpose

This app was created to practice:

* Clean Architecture (Data, Domain, Presentation layers)
* Dependency Injection using Hilt
* Room Database
* Kotlin Coroutines and Flow
* Jetpack Compose UI
* MVVM pattern
* Feature-based modular thinking

## Features Implemented

* Focus session tracking
* Expense tracking
* Habit tracking
* Rule-based insights
* Dashboard combining multiple data sources

## Tech Stack

* Kotlin
* Jetpack Compose
* Hilt (Dependency Injection)
* Room Database
* Coroutines & Flow
* MVVM + Clean Architecture

## Note

This project is primarily built for **learning, experimentation, and portfolio demonstration** to improve Android architecture and system design skills.
